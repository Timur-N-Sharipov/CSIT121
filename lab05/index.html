All about Hyper Text Markup Language

My updated webpage! I learned about the basics of HTML, how to set up the file, the names of different tags, what to use them for, and the general semantic structure a webpage should have. Now I will be utilizing what I learned to build this blog like webpage.

What I learned about HTML

  1. Making an ordered and an unordered list.
  2. Creating hyperlinks that take you to another page.
  3. Semantic HTML used to create the structure of a page
  4. Different heading sizes

All about Cascading Style Sheets

I am back to update my blog post but this time around I will utilize my new found knowledge of Cascading Style Sheets, or CSS for short. CSS is a language that is used alongside HTML to style and layout content, this is how you can make your webpage visually pleasing to look at and browse through.

What I learned about CSS

  1. How to add CSS to HTML documents using inline style, internal(embedded) and external stylesheets
  2. CSS specificity, as well as the ID and Class selectors
  3. How to alter HTML elements, such as their color, font size, font weight and more!

I learned that a good practice when developing a webpage is to build it with responsive design in mind. As individuals will be using different devices sizes to visit you webpage. CSS3 has the perfect solution with media queries
