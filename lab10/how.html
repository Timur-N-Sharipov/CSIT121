Web Cryptography Usage

By using Web Cryptography API a website can encrypt a password before sending it to a server, encrypt a message so that only the person who it's intended for can decrypt and read it. Cryptography encryption relies on algorithms to scramble the readable data into unrecognizable, unreadable text or a string of mixed characters, this cyphered version is refered to as ciphertext, which later can be decrypted into the original readable data. This is done using complex mathematical algorithms

Here is a list of ways it is used
  • Multi-factor Authentication
  • Protected Document Exchange
  • Secure Messaging
  • Data Integrity Protection
