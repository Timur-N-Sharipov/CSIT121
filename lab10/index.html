Cryptograhy is a practice of using mathematican and coding algorithms to protect data so that it is only accessible to authorized individuals. The Web Cryptography Application Programming Interface (API) is a JavaScript based standard that is used in modern browsers to perform encryption, decryption, hashing, and verification. Web Cryptography API is created by the World Wide Web Consortium (W3C) published in January 26 2017 and is a modern standard. In other words it is a standard set so that your data can be safely and securely transfered through the web while maintaining its privacy, security and integrity, protecting your sensitive information from hackers and cybercriminals.
